# project-2
Project 2: SOC – Endpoint Detection & Response (EDR) Grid
I worked on a Security Operations Center (SOC) project focused on building an Endpoint Detection & Response (EDR) system for real-time threat monitoring and automated response.
🔍 Description:
This project simulates an enterprise-level SOC environment where systems are continuously monitored for suspicious activities such as brute-force attacks, file modifications, and malware behavior.
⚙️ Key Features:
• File Integrity Monitoring (FIM) to detect unauthorized file changes
• Real-time log monitoring using Wazuh
• Detection of suspicious activities and anomalies
• Automated active response (e.g., blocking malicious IPs)
• Threat mapping using MITRE ATT&CK framework
🛠 Tech Stack:
Wazuh, Sysmon, Linux, Windows Server
🎯 Objectives:
• Understand real-time threat detection
• Implement log monitoring and alerting systems
• Automate response to security incidents
⚡ Highlights:
• Detected file changes within seconds
• Simulated brute-force attacks and blocked attacker IPs
• Visualized attack patterns and threat behavior
🌐 Future Scope:
• Advanced threat hunting
• Integration with SIEM dashboards
• Enhanced automation and response systems
